## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation
```bash
# Clone the repository
git clone https://github.com/Ornerd/ticket-mgt-app-react.git
cd ticket-mgt-app-react

# Install dependencies
npm install

# Run development server
npm run dev
```
